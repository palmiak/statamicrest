# How to connect headless Statamic with Astro

This is a repository used in this [guide](https://buddy.works/guides/statamic-rest-api).

## Instalation

1. Fill out the `.env.example` file and change it's name to `.env`.
2. Run

```bash
npm install
npm run build
```
